Huffman Compression

A console based application that allows users to compress files without losing data. Uses the famous Huffman Coding Algorithm.

Live Demo Available upon request

Project Overview

An application built using Python which provides lossless data compression using the Huffman Algorithm. Through compression, users can upload files which would usually be considered too large to upload.

Similar to how zip works on computers. Compatible on all major Operating Systems.
